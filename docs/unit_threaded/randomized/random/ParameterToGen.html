    Alias ParameterToGen

    A template that turns a T into a Gen!T unless T is already a Gen or no Gen for given T is available.

    alias ParameterToGen(T) = T;

    Example

    alias GenInt = ParameterToGen!int;

static fun(int i)
{
    assert(i == 1337);
}

GenInt a;
a.value = 1337;
fun(a);

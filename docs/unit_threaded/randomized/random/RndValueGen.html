    Struct RndValueGen

    This type will generate a Gen!T for all passed T.... Every call to genValues will call gen of all Gen structs present in values. The member values can be passed to every function accepting T....

    Example

    import unit_threaded.randomized.gen: Gen;
import std.random: Random;

auto rnd = Random(1337);
auto generator = (&rnd).RndValueGen!(["i", "f"],
                                     Gen!(int, 0, 10),
                                     Gen!(float, 0.0, 10.0));
generator.genValues();

static fun(int i, float f)
{
    import std.conv: to;
    assert(i >= 0 && i <= 10, i.to!string);
    assert(f >= 0.0 && f <= 10.0, f.to!string);
}

fun(generator.values);
